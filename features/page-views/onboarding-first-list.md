**Route:** `onboard/first-list/{listId}`
**Group:** Onboarding
**Status:** active
**Last reviewed:** 2026-06-08
**Index:** [Page views](../PAGE_VIEWS.md)

What a user expects to see or do here:

- Name the list and pick at least 3 people to finish
- Use the same configuration controls as the real Edit-list screen
- Finish ("Done") or add another list
- Clear gating: can't finish without a name + 3 members
