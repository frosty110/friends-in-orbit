# Contact detail

**Route:** `contact/{contactId}`
**Group:** Contact
**Status:** active
**Last reviewed:** 2026-06-08
**Index:** [Page views](../PAGE_VIEWS.md)

What a user expects to see or do here:

- See photo, name, number, and which lists they're on
- Read call stats: last called, total calls, avg length, longest gap, usual answer time
- Scroll recent call history with direction + duration
- Add and read notes (jump straight to the note input from some entry points)
- Pause the person for a chosen duration, or unpause
- Ignore them, or add them to more lists
- Re-link if they were unlinked from a device contact
